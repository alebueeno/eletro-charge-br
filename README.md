# Eletro Charge BR - Landing Page

**Acesso ao projeto ao vivo:** https://alebueeno.github.io/eletro-charge-br/

## Sobre o Projeto
Este é um projeto **fictício** desenvolvido exclusivamente para fins acadêmicos como trabalho prático da disciplina de Programação Front-End/Análise e Desenvolvimento de Sistemas. 

A **Eletro Charge BR** foi idealizada (originalmente para um trabalho de Administração) como uma empresa de infraestrutura sustentável focada em mobilidade elétrica. O objetivo deste projeto de Front-End foi transformar essa regra de negócio em uma Landing Page funcional, priorizando a narrativa visual, estruturação semântica e a experiência do usuário (UX).

## Tecnologias Utilizadas
Como requisito da avaliação, este projeto foi construído **exclusivamente** com as linguagens base da web estática:
* **HTML5:** Estruturação semântica (uso de tags como `<header>`, `<section>`, `<footer>`, etc).
* **CSS3:** Estilização, responsividade, CSS Grid, Flexbox e animações puras (hover, smooth scroll, parallax).
* **FontAwesome:** Biblioteca de ícones em fonte.
* **Google Fonts:** Tipografia (família *Inter*).

## Arquitetura do Código
Para demonstrar boas práticas de desenvolvimento, o CSS foi refatorado e modularizado, separando responsabilidades:
* `global.css`: Variáveis globais (cores, fontes) e classes utilitárias.
* `layout.css`: Componentes estruturais repetitivos (Cabeçalho, Botões, Rodapé).
* `home.css`: Estilos específicos da página (Grid de produtos, Timeline).
